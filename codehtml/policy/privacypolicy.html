Kebijakan Privasi

Terakhir diperbarui: 15/11/2024

Kami menghargai privasi Anda dan berkomitmen untuk melindunginya. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi yang Anda berikan saat menggunakan aplikasi kami. Kami juga menjelaskan bagaimana kami mematuhi persyaratan yang ditetapkan oleh Google terkait dengan penggunaan berbagai API mereka, seperti YouTube API, Google Drive API, dan layanan Google lainnya.

1. Informasi yang Kami Kumpulkan

Kami mengumpulkan informasi yang Anda berikan kepada kami melalui aplikasi ini, termasuk:

  • Informasi Akun Google: Kami akan meminta izin untuk mengakses akun Google Anda melalui OAuth 2.0 untuk menggunakan berbagai layanan Google seperti Google Drive, YouTube, dan API lainnya. Ini memungkinkan kami untuk mengelola data Anda di berbagai layanan Google yang diizinkan.
  • Data Akses API: Kami mengumpulkan data terkait interaksi Anda dengan API yang digunakan oleh aplikasi ini, termasuk permintaan API, tanggapan API, dan status otentikasi.
  • Data File: Jika Anda mengunggah atau mentransfer file ke Google Drive atau layanan Google lainnya melalui aplikasi ini, data tersebut akan diproses dan disimpan sesuai dengan kebijakan privasi Google.

2. Penggunaan Data

Data yang kami kumpulkan digunakan untuk tujuan berikut:

  • Pengelolaan Layanan Google: Untuk memungkinkan Anda mengelola file Anda di Google Drive, mengelola saluran YouTube Anda, dan menggunakan layanan Google lainnya yang Anda pilih.
  • Keamanan dan Pemeliharaan: Untuk memantau penggunaan aplikasi dan menjaga keamanan sistem kami.

3. Bagaimana Kami Melindungi Data Anda

Kami mengambil langkah-langkah untuk melindungi data Anda dengan menggunakan enkripsi dan teknologi keamanan yang sesuai. Kami tidak akan membagikan data pribadi Anda dengan pihak ketiga tanpa izin Anda, kecuali jika diperlukan untuk memenuhi kewajiban hukum atau untuk melindungi hak-hak kami.

4. Kebijakan Data dari Pihak Ketiga

Kami hanya menggunakan API yang disediakan oleh Google untuk mengakses data di akun Google Anda, seperti Google Drive, YouTube, dan layanan Google lainnya. Kami tidak akan mengakses data atau informasi pribadi Anda tanpa izin eksplisit dari Anda.

  • Layanan Google: Data yang Anda unggah atau unduh melalui aplikasi ini akan disimpan di layanan Google seperti Google Drive, sesuai dengan kebijakan privasi Google.
  • Google API: Aplikasi ini dapat mengakses API Google untuk memungkinkan penggunaan fitur-fitur terkait, seperti pengelolaan file atau saluran YouTube.

5. Penyimpanan Data

Semua data yang diproses melalui aplikasi ini akan disimpan di layanan Google yang Anda pilih (misalnya, Google Drive, Google Cloud Storage) sesuai dengan kebijakan privasi Google. Kami tidak akan menyimpan data Anda di server kami sendiri.

6. Akses dan Kontrol Pengguna

Anda dapat mencabut izin akses aplikasi ini ke akun Google Anda kapan saja melalui pengaturan akun Google Anda. Setelah pencabutan izin, aplikasi ini tidak akan dapat mengakses data Google Anda.

7. Persetujuan Pengguna

Dengan menggunakan aplikasi ini, Anda setuju untuk memberikan izin aplikasi untuk mengakses akun Google Anda melalui OAuth 2.0, dan untuk menggunakan data yang diperlukan sesuai dengan Kebijakan Privasi ini. Jika Anda tidak setuju, Anda tidak dapat menggunakan fitur-fitur yang terkait dengan API Google.

8. Perubahan pada Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Setiap perubahan akan diumumkan di halaman ini dengan tanggal pembaruan yang baru. Kami mendorong Anda untuk memeriksa halaman ini secara berkala agar tetap mengetahui bagaimana kami melindungi informasi Anda.

9. Kontak

Jika Anda memiliki pertanyaan atau kekhawatiran mengenai Kebijakan Privasi ini atau penggunaan data Anda, silakan hubungi kami di Email Kami.
